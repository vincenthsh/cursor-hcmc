## Overview

Cacophony is a multiplayer music battle game where 3-8 players compete to create the funniest AI-generated songs by matching music genres (vibe cards) with humorous lyrics (lyric cards).

## Game Setup

### Creating a Room

1. Click **"Create Room"** on the home page
2. Share the 6-character room code with friends
3. Wait for players to join (minimum 3 players)
4. Click **"Start Game"** when ready

### Joining a Room

1. Click **"Join Game"** on the home page
2. Enter the 6-character room code
3. Choose your username
4. Wait for the host to start the game

## Game Roles

### The Producer (1 per round)

**Responsibilities:**
- Receives the vibe card (e.g., "A sad Country song about ______")
- Listens to all artist submissions
- Votes for the funniest song

**Note**: The Producer role rotates each round based on join order, so everyone gets a turn!

### The Artists (all other players)

**Responsibilities:**
- View their hand of 5 lyric cards
- Select one lyric card to answer the vibe card
- Submit for AI song generation

**Goal**: Make the Producer laugh with creative combinations!

## How a Round Works

### Step 1: Vibe Card Revealed
- The Producer's vibe card is shown to all players
- Example: "A heartbreaking Country ballad about ______"

### Step 2: Artists Select & Submit (60 seconds)
- Artists choose one lyric card from their hand
- Example cards: "pizza delivery gone wrong", "existential dread at the DMV"
- Click "Submit Card" when ready

### Step 3: Songs Generate
- AI (Suno) creates a song for each submission
- Combines: Vibe card + Lyric card
- Example result: "A heartbreaking Country ballad about pizza delivery gone wrong"

### Step 4: Producer Listens & Votes
- Producer plays each generated song
- Chooses the funniest one
- Winner is announced

### Step 5: Scoring
- Winning artist gets 1 point
- Producer role rotates to next player
- New round begins

## Winning the Game

- Game continues for a set number of rounds (default: 5)
- Player with the **most points** at the end wins
- Points are only earned as an Artist (not as Producer)

## Card System

### Vibe Cards (103 total)

Genre/style cards that set the musical theme:
- "A heartbreaking Country ballad about ______"
- "An aggressive Death Metal song destroying ______"
- "A cringey Corporate Jingle selling ______"
- And 100 more wild combinations!

### Lyric Cards (583 total)

Pre-written humorous phrases that answer the vibe card:
- "my therapist's therapist"
- "goblin mode activated"
- "existential dread at the DMV"
- "Hot Girl Summer (I failed)"
- And 579 more ridiculous options!

**Note**: Lyric cards are complete phrases - no blank-filling required!

## Technical Notes

### Song Generation Time
- Each song takes approximately 30-60 seconds to generate
- Generation happens in parallel for all submissions
- Progress bars show real-time status

### Browser Compatibility
- Best on Chrome, Firefox, Safari (latest versions)

## Frequently Asked Questions

**Q: Can I play solo?**
A: No, you need at least 3 players total.

**Q: What happens if someone disconnects?**
A: They're marked as inactive. The host can kick them and continue.

**Q: Can I rejoin if I refresh?**
A: Yes, if you're using the same browser/device. Your player ID is saved in local storage.

**Q: How are songs generated?**
A: Using Suno AI's music generation API. Songs are unique and created in real-time.

**Q: Is there voice chat?**
A: Not built-in. We recommend using Discord, Zoom, or similar for voice.

**Q: Can I save my favorite songs?**
A: Not currently - songs are temporary. Feature planned for future!
